# Florida Septic — Strategy Report

Go-to-market evaluation and recommended launch plan for **Florida Septic**, a lead-generation marketplace serving Florida homeowners affected by the state's septic upgrade legislation (SB 712 / HB 1379 / HB 1417).

The report compares four launch approaches and recommends a phased "Credibility-First Lead Engine" — an 8–9 page MVP with a Grant Calculator differentiator, built across three sprints over 6–8 weeks. Brevard County is the wedge market, with phased expansion across the IRL Protection Program counties and Florida's Springs Belt.
